# My custom Neovim configuration

run: `git clone https://github.com/Marcus441/NvChad-Config ~/.config/nvim && nvim`
`:MasonInstallAll`

### Dependencies:

Required:

- NVIM Stable v0.10.0+ (Nightlies might work)
- NvChad v2.5
- Python 3.11.7+ & pip
- pip: `pynvim==0.4.3+`
- Node.js v22+
- npm: `neovim@4.10.1+`

Recommended:

- Cargo 1.74.1+
